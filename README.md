# 五張撲克牌（含下注）

## 專案說明

這個專案是用 C# WinForms 做的一個五張撲克牌遊戲。
原本只有基本發牌跟判斷牌型，後來有再加下注功能，還有做成單人跟雙人模式。

雙人模式是兩個人輪流玩，會顯示目前是誰的回合，也會各自算錢。

## 功能

* 發五張牌
* 可以換牌
* 判斷牌型（像順子、同花之類）
* 可以下注
* 依照牌型給不同倍率
* 單人模式
* 雙人模式（輪流下注）

---

## 下注怎麼算

基本上就是：

```id="simplecalc"
中獎 = 押注金額 × 對應倍率
```

如果沒中就會扣掉押的錢。
