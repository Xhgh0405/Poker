# 五張撲克牌

## 專案簡介
這是 Windows Forms 五張撲克牌練習專案，功能包含發牌、換牌、判斷牌型、下注、賠率計算。

## 新增功能
- 單人模式：一位玩家下注並計算目前金額。
- 雙人模式：玩家 1 與玩家 2 輪流下注。
- 畫面會顯示目前玩家、玩家 1 金額、玩家 2 金額。
- 依照牌型賠率計算中獎金額。

## 操作方式
1. 選擇單人或雙人模式。
2. 輸入押注金額並按「下注」。
3. 按「發牌」。
4. 點選不想保留的牌，牌會變成背面。
5. 按「換牌」。
6. 按「判斷牌型」，系統會計算中獎金額。
7. 若為雙人模式，系統會自動換下一位玩家下注。
